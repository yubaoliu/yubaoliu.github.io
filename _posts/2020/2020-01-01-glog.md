# Config

- CMakeLists.txt
```sh
find_package(Glog REQUIRED)
include_directories(BEFORE ${GLOG_INCLUDE_DIRS})
```
- Include header
```cpp
#include <glog/logging.h>
google::InitGoogleLogging(argv[0]);
```

Example:
 ```cpp
 DLOG(INFO)<<"This is Dlog";
 LOG(INFO)<<"This is LOG";
 ```
