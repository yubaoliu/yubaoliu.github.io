1 Install Turtlebot ROS Packages

sudo apt install ros-kinetic-turtlebot-*

2 Network Configuration

Config Turtlebot robot as the master machine. My PC as the slave machine.

2.0.1 Install ssh

sudo apt-get install openssh-server
ps -e|grep ssh

ssh [user_name]@[ROBOT_IP]

2.0.2 Network Time Protocol

sudo apt-get install chrony
sudo ntpdate ntp.ubuntu.com

ntpdate -q other_computer_ip

2.0.3 Add host name

sudo vim /etc/hosts

Add: 

133.15.23.26    yubao-pc
133.15.23.206   kobuki

Restart Network 

sudo /etc/init.d/networking restart

2.0.4 Modify bash file to config ROS

Kobuki: 

echo export ROS_MASTER_URI=http://IP_OF_TURTLEBOT:11311 >> ~/turtlebot/devel/setup.sh
echo export ROS_HOSTNAME=IP_OF_TURTLEBOT >> ~/turtlebot/devel/setup.sh

Remote PC Setup: 

echo export ROS_MASTER_URI=http://IP_OF_TURTLEBOT:11311 >> ~/.bashrc
echo export ROS_HOSTNAME=IP_OF_PC >> ~/.bashrc

or add these lines to ~/.bashrc 

export ROS_MASTER_URI=http://[kobuki-ip]:11311
export ROS_HOSTNAME=<PC_IP>
export ROS_IP=<PC_IP>

Example: 

export ROS_MASTER_URI=http://kobuki:11311
export ROS_HOSTNAME=133.15.23.26
export ROS_IP=133.15.23.26

3 Verify from TurtleBot to Remote PC

3.0.1 Hello Message

On remote pc run: 

rostopic pub -r10 /hello std_msgs/String "hello"

On TurtleBot run 

rostopic echo /hello

3.0.2 Turtlesim Example

koboki: 

roscore

rosrun turtlesim turtlesim_node

PC: 

rosrun turtlesim turtle_teleop_key

yubao@yubao-Z370M-S01:~$ rosrun turtlesim turtle_teleop_key
Reading from keyboard
---------------------------
Use arrow keys to move the turtle.

4 Start Turtlebot

kobuki: 

roslaunch turtlebot_bringup minimal.launch --screen

Remote: 

roslaunch turtlebot_teleop keyboard_teleop.launch

rosrun turtlesim draw_square

检查底座信息 

rqt -s kobuki_dashboard
