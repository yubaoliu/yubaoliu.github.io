* Compile

#+BEGIN_SRC sh
    git clone git@github.com:yubaoliu/caffe.git
    sudo apt-get install libprotobuf-dev libleveldb-dev libsnappy-dev libopencv-dev libhdf5-serial-dev protobuf-compiler
    sudo apt-get install --no-install-recommends libboost-all-dev
    sudo apt-get install liblmdb-dev
    sudo apt install libboost1.55-all-dev
#+END_SRC

** Use Cmake

#+BEGIN_SRC sh
    mkdir build
    cd build
    cmake ..
    make all
    make install
    make runtest
#+END_SRC

* Matlab
  :PROPERTIES:
  :CUSTOM_ID: matlab
  :END:

Install MATLAB, and make sure that its mex is in your =$PATH=.

Caffe's MATLAB interface works with versions 2015a, 2014a/b, 2013a/b,
and 2012b.
