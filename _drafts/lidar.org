* 2D LIDAR
** HOKUYO

How to use:
- http://wiki.ros.org/hokuyo_node/Tutorials/UsingTheHokuyoNode
- http://wiki.ros.org/urg_node



![HOKUYO](http://wiki.ros.org/urg_node?action=AttachFile&do=get&target=hokuyo.jpg)

- Instal Ros driver:

#+begin_src bash
sudo apt install ros-kinetic-urg-node
#+end_src

- Configuring the Hokuyo

#+begin_src bash
$ls -l /dev/ttyACM0
crw-rw---- 1 root dialout 166, 0  6月  2 17:52 /dev/ttyACM0
#+end_src

#+begin_src bash 
crw-rw-XX- 1 root dialout 166, 0 2009-10-27 14:18 /dev/ttyACM0
#+end_src

If XX is rw: the laser is configured properly.

If XX is --: the laser is not configured properly and you need to:

#+begin_src bash
$ sudo chmod a+rw /dev/ttyACM0
#+end_src

- Run node
#+begin_src 
roscore
rosrun urg_node urg_node
rostopic type /scan
rosmsg show sensor_msgs/LaserScan

#+end_src

- rvize
#+begin_src 
rosrun rviz rviz
#+end_src
- Fixed frame: laser
- Add LaserScan
